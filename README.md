# Task 5: Personal Loan Acceptance Prediction

## Objective
Predict whether a customer is likely to accept a personal loan offer.

## Approach
- Explored customer demographic and financial data.
- Performed data preprocessing and encoding.
- Conducted Exploratory Data Analysis (EDA).
- Trained a Logistic Regression classification model.
- Evaluated performance using accuracy and confusion matrix.
- Extracted business insights from customer behavior patterns.

## Results and Insights
- Customer age, job type, and marital status influenced loan acceptance.
- Certain customer segments showed higher acceptance rates.
- The model successfully predicted customer responses to loan offers.
- The findings can help banks improve targeted marketing campaigns and increase conversion rates.
